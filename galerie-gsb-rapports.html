Projet GSB Rapports
Technologies C# · WinForms · SQL Server · ADO.NET
Type Application desktop Windows

À propos du projet

Application desktop Windows développée en C# avec WinForms pour le laboratoire Galaxy Swiss Bourdin. Elle permet la gestion et l'affichage des rapports de visite des visiteurs médicaux, avec une connexion à la base de données SQL Server via ADO.NET.
